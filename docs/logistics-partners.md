  - Rembourrage : Papier, mousse (articles fragiles)
  - Étiquette lisible : Visible sur le colis

  - Détails : Produit, prix, date livraison prévue
  - Instruction COD si applicable

  - Pile Amana

  - Pile Coursier
- [ ] **11h00** : Dépôt auprès des partenaires
  - Amana avant 11h (pour même jour)
  - Glovo via app instantanée
  - Coursier par appel/WhatsApp
- [ ] **11h30** : Mise à jour système
  - Statut : "Package shipped ✓"
  - Numéro suivi du partenaire
- [ ] **12h00** : Notification client WhatsApp
  - Message : "Votre commande est en route! Vous la recevrez demain. 📦"
  - Inclure numéro de suivi si disponible

---

### Before Delivery (1 heure avant)

**Préparation du client :**

- [ ] **-1h** : Appel client automatique ou manuel
  - Message : "Bonjour! Votre livreur arrive dans 1 heure, soyez prêt(e)! 📞"
  - Vérifier numéro correct
- [ ] **-45 min** : Confirmation d'adresse
  - Repérer l'adresse exacte
  - Coordonnées GPS si possible
- [ ] **-30 min** : Vérifier mode de paiement
  - Cash : Client a l'argent prêt?

## 6. PROBLEM RESOLUTION

### Matrice de résolution SLA

| Problème | SLA | Solution | Responsable |
|----------|-----|----------|-------------|
| Adresse incorrecte | 2h | Redirection ou renvoi (notre coût) | Operations |
| Client indisponible | 1 jour | 3 tentatives, puis hold ou retour | Partenaire + Ops |
| Article endommagé | 4h | Remboursement complet + excuses | Support |
| Livraison tardive | 24h | Crédit de 10% pour prochaine commande | Support |
| Paiement refusé | 4h | Nouvelle tentative ou reprogrammer | Operations |
| Perte du colis | 24h | Réclamation auprès partenaire + remboursement | Support |

---

### Detailed Problem Handling

**PROBLÈME 1 : Adresse Incorrecte**
- Détection : Client ou livreur signale adresse non trouvée
- Délai SLA : 2h
- Actions :
  1. Appel client immédiat pour adresse correcte
  2. Si trouvable : Redirection du livreur (gratuit)
  3. Si pas trouvable : Retour au magasin + renvoi demain (notre coût)
  4. Compensation : +1 crédit client pour prochaine commande
- Coût pour nous : 35 MAD (double livraison)

**PROBLÈME 2 : Client Indisponible**
- Délai SLA : 1 jour complet
- Actions :
  1. Tentative 1 : Appel 30 min avant
  2. Tentative 2 : Livreur sonne à la porte (heure prévue)
  3. Tentative 3 : Appel client après demi-échec
  4. Après 3 échecs : Retour à la gare/entrepôt + stockage 48h
  5. Suivant : Re-coordination avec client pour nouveau créneau
- Coût pour nous : 0 (partenaire reprend)

**PROBLÈME 3 : Article Endommagé**
- Délai SLA : 4h
- Actions :
  1. Photo du dégât prise par client
  2. Vérification de responsabilité (notre packaging vs dégât transport)
  3. Si notre faute : Remboursement complet + nouvel article gratuit
  4. Si transport : Réclamation contre partenaire (remboursement client de notre poche initialement)
  5. Message client : "Sincères excuses! Remboursement traité aujourd'hui."
- Coût pour nous : 100% du produit

**PROBLÈME 4 : Livraison Tardive**
- Délai SLA : 24h (reconnaissance du délai)
- Actions :
  1. Vérifier si délai garanti était promis
  2. Si oui ET limité à 4h de retard : Crédit de 10% sur prochaine commande
  3. Si oui ET plus de 4h : Remboursement complet de la livraison + 10% crédit
  4. Message client : "Désolé pour le délai! Voici un crédit de X MAD pour votre prochaine commande."
- Coût pour nous : 10-50 MAD selon sévérité

**PROBLÈME 5 : Paiement Refusé (COD)**
- Délai SLA : 4h
- Actions :
  1. Livreur rapporte le refus de paiement
  2. Appel client immédiat : "Le paiement n'a pas abouti, quand puis-nous réessayer?"
  3. Option A : Programmer nouvelle livraison (gratuit)
  4. Option B : Transfer bancaire direct (si client accepte)
  5. Délai max avant retour : 48h
- Coût pour nous : 35 MAD (double livraison si renvoi)

**PROBLÈME 6 : Perte du Colis**
- Délai SLA : 24h
- Actions :
  1. Confirmation avec partenaire (traçabilité)
  2. Déclaration de perte officielle
  3. Réclamation auprès du partenaire
  4. Pendant ce temps : Remboursement client immédiat (notre argent)
  5. Récupération de remboursement partenaire après (2-4 semaines)
- Coût pour nous : 100% du produit + 35 MAD livraison

- 30 Casablanca @ 50 MAD = 1,500 MAD
