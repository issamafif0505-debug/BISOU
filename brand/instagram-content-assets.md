# VISUAL DESIGN GUIDELINES FOR CAPTIONS

## Color Coding System:
- 🌟 Gold/highlights = Product benefits
- ❤️ Heart/red = Emotional benefits
- ✨ Sparkles = Quality/premium feel
- 🔥 Fire = Urgency/sales
- 💎 Diamond = Luxury/prestige

## Formatting Rules:
1. Always start with emoji (eye-catching)
2. Use line breaks (easier to read on mobile)
3. Bold the price in ALL CAPS
4. Use checkmarks (✓) for benefits
5. Use arrows (→) for comparisons
6. End with CTA (call to action)
7. Include 15-20 relevant hashtags

## Best Practices:
- Keep first 3 lines under 100 characters (hook phase)
- Use 2-3 emojis per sentence maximum
- Break text into short sentences
- Use active voice
- Include specific numbers (229 MAD, not "affordable")
- Always include WhatsApp/link CTA
- Tag location if relevant
